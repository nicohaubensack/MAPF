The code is to be executed in the MAPF.py file. There, these three main functions can be executed:
1. Figures.solve_instance(inst,solvers = ["PR","FB"]), this function solves the instance and displays a visualization of the solution. Single solvers can be removed. This might be useful if the instance is too larger for the FB Solver.
2. Figures.display_instance(inst), this function displays the instance in a grid format
3. Figures.display_TEN(inst), this function displays the time-expanded network of the instance on its optimal makespan

Also, all figures and tables of the thesis can be generated by the two functions at the bottom. Keep in mind that some of them might take a long time to finish.
